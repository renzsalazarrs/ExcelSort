Preparation
1. Download ExcelSort.dll and place it on a location of your choice.

Steps
1. Open C++ Task.xlsm
2. Go to Developer Tab and Select Visual Basic.
3. In wksTable sheet module, modify library path for DLL access.
   -> Lib "C:\your_path_here\ExcelSort.dll
4. Debug -> Compile once.
5. Go to wksTable sheet and you can now sort successfully.
